📱 ENDPOINTS PRINCIPALES
================================================================================

AUTENTICACIÓN:
POST   /auth/register    - Crear usuario
POST   /auth/login       - Obtener token JWT
GET    /auth/me          - Datos del usuario

ANÁLISIS:
POST   /analyze          - Analizar riesgo de texto
POST   /embedding        - Generar embedding vectorial

GEOLOCALIZACIÓN:
POST   /geo              - Guardar ubicación
GET    /geo/nearby       - Buscar eventos cercanos
GET    /geo/user/{id}    - Ubicaciones de usuario

HISTORIAL:
POST   /comments         - Registrar evento
GET    /comments/{id}    - Historial de usuario

MONITOREO:
GET    /health           - Estado del sistema
GET    /docs             - Documentación Swagger
